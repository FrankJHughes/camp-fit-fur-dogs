# CI/CD Guides

This folder contains documentation for the **continuous integration and continuous deployment** pipelines that support the CampFitFurDogs project.

These guides explain how GitHub Actions workflows are structured, how dependencies flow, and how contributors should interact with CI/CD.

- **[CI Dependency Graph](../ci-dependency-graph.md)**  
  How workflows depend on each other and how changes propagate.

- **[Action README Template](../action-readme-template.md)**  
  Template for documenting new GitHub Actions.

---

# Purpose

These guides help developers:

- Understand how CI/CD is structured  
- Diagnose pipeline failures  
- Add or modify GitHub Actions  
- Maintain consistency across workflows
