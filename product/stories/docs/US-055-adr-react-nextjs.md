# US-055 — ADR-0012: Frontend Technology (React/Next.js)

## User Story

As a **contributor**, I want a documented architecture decision for the frontend framework, so that all team members understand the technology choice and its rationale.

## Context

The team has informally decided on React with Next.js and TypeScript for the frontend. This ADR formalizes the decision, captures alternatives considered, and documents consequences for build tooling, deployment, and API integration.

## Scope

- One ADR file in `docs/adr/` following the existing template.
- Updated ADR index in `docs/adr/README.md`.
