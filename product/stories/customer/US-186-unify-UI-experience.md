# US‑186 — Unify UI Experience

## Story Grammar  
As an **owner**, I should experience a **cohesive, polished, and consistent user interface** so that **the product feels professional, intentional, and showcase‑ready**.

## Intent  
The current UI is fragmented across pages, components, and flows. This story unifies layout, styling, navigation, and interaction patterns into a single, coherent experience suitable for demonstration to potential employers.

## Acceptance Criteria  
- A unified layout applied across all pages (header, footer, navigation, spacing, typography)  
- Consistent design system tokens (colors, spacing, typography, components)  
- Login → dashboard → navigation flow visually cohesive  
- Remove or restyle placeholder pages to match the unified design  
- Mobile‑responsive baseline established  
- No mismatched components, inconsistent spacing, or divergent styles  
- All pages use the same design primitives (buttons, inputs, cards, alerts)  
- Visual polish suitable for a professional showcase
