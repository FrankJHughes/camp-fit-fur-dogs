# Planning Runbook

## Intent
Create a living document (docs/runbook.md) that captures the end-to-end planning ceremony — from writing product specs through YAML authoring, script execution, and board verification.

## Value
Enables any team member to run the full planning workflow independently, reducing bus-factor risk and onboarding time.

## Acceptance Criteria
- [ ] docs/runbook.md exists with step-by-step ceremony instructions
- [ ] Covers: product spec authoring, YAML creation, schema rules, script execution, board verification
- [ ] Includes troubleshooting section for common failures
- [ ] Cross-referenced from CONTRIBUTING.md and README.md
